# Customer Churn Prediction

## Problem
Predict which telecom customers will churn (cancel service).

## Dataset
- 7043 customers
- 21 features
- 27% churn rate

## Key Finding
Month-to-month customers churn 14x more than two-year contract customers.

## Model
- Algorithm: Logistic Regression
- Accuracy: 81.5%
- Precision: 86% (retained), 68% (churned)

## Tech Stack
Python, Pandas, NumPy, Scikit-learn
